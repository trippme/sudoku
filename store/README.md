# Store assets

Screenshots captured from the app (Pixel 5 emulator, 1080×2340), ready to
upload to the stores.

- `screenshots/play/` — native **1080×2340**, ready for **Google Play**.
- `screenshots/appstore-6.7in/` — **1290×2796**, the **App Store 6.7"** size
  (iPhone 15/16 Pro Max etc.). Same images scaled; aspect ratio matches so
  there's no distortion.

## The four shots (suggested order + captions)

1. **01-home** — *"Pick up where you left off, or start any difficulty."*
   The home menu: daily puzzle, play-by-number, four difficulties, inbox.
2. **02-game** — *"Tap a number to highlight it everywhere — clean, fast input."*
   A game in play with the selected digit (green) and same-number highlighting.
3. **03-hint** — *"Stuck? Hints explain the technique, not just the answer."*
   The explanatory hint dialog (a Naked Single, with the reasoning).
4. **04-leaderboard** — *"Race friends on the exact same puzzle."*
   A game's leaderboard, ranked by fewest hints then fastest time.

## Notes
- The leaderboard shot shows live demo data (Alice/Bob). Before publishing you
  may want a cleaner capture with your own profile set (no "set your email"
  banner) — easy to grab on a real device.
- Other useful sizes if a listing asks: App Store 6.5" is **1242×2688**; Play
  also accepts the native size directly.
